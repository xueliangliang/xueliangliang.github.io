How?

How was this website built? I followed some simple principles and make use of a lot of existing open source tools

The principles are:

  • Other People's ____

    • I used a lot other people's library/tool

  • KISS Principle

    • Keep it simple and stupid. I seperate the website into different modules and layers, each module and layer only have a very simple responsibility. I used GulpJs to manage the packaging and building process

  • Kill two birds with one stone

    • Before this assignment, I didn't use GulpJs, I just took chance to learn how to use GulpJs to manage html frontend workflow.

  • Think Out of Box

    • To think out of box, we need know where the box is. Initially I just want to use a tranditional website to display all the information. I happened to view TimeLineJs's website, I found it is more informative to show my portfolio. So now you see the WOW effect.

The tools used:

  • SASS
  • TimelineJS
  • SWIG Template
  • PhotoPile
  • GulpJs
